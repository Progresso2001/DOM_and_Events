# DOM and Events
DOM_Events in Js

O código fornecido demonstra o uso de vários eventos de formulário em JavaScript, junto com alguns ouvintes de eventos e funções de gerenciamento de eventos relacionados. Aqui está uma breve explicação do código:

Evento de envio de formulário: O evento de envio é acionado quando o formulário é submetido. No código, um ouvinte de eventos é adicionado ao elemento do formulário e, quando o formulário é submetido, a função de gerenciamento de eventos impede o comportamento padrão de envio do formulário usando e.preventDefault(), e registra os dados do formulário, incluindo os valores dos campos de nome de usuário, e-mail e gênero, no console.
Evento de reset do formulário: O evento de reset é acionado quando o formulário é reiniciado. No código, um ouvinte de eventos é adicionado ao elemento do formulário e, quando o formulário é reiniciado, a função de gerenciamento de eventos registra uma mensagem no console indicando que os dados do formulário foram apagados.
Evento de alteração: O evento de mudança é acionado quando o valor de um campo de formulário, como um elemento select, é alterado. No código, um ouvinte de eventos é adicionado ao elemento de seleção de curso, e quando o curso selecionado é alterado, a função de gerenciamento de eventos registra o valor do campo selecionado no console.
Propriedade verificada: A propriedade verificada é usada para determinar se uma caixa de seleção ou um botão de seleção está marcado ou não. No código, um ouvinte de eventos é adicionado ao elemento da caixa de seleção, e quando a caixa de seleção está marcada ou desmarcada, a função de gerenciamento de eventos registra uma mensagem no console de acordo.
Evento de entrada: O evento de entrada é acionado quando o valor de um campo de entrada, como uma entrada de texto ou área de texto, é alterado. No código, um ouvinte de eventos é adicionado ao elemento de entrada do nome de usuário e, quando a entrada do nome de usuário é alterada, a função de gerenciamento de eventos registra o valor alterado no console.
Eventos de foco e desfoque: O evento de foco é acionado quando um elemento recebe foco, e o evento de desfoque é ativado quando um elemento perde o foco. No código, ouvintes de eventos são adicionados ao elemento de entrada com nome de usuário, e quando o elemento de entrada recebe foco, a cor da borda muda para laranja, e quando perde o foco, a cor da borda volta a ser preta.
Em resumo, o código fornecido demonstra o uso de eventos de formulário em JavaScript, incluindo submeter, reset, alterar, verificar, input, focus e blur eventos, juntamente com seus respectivos ouvintes de eventos e funções de gerenciamento de eventos. Esses eventos e funções de gerenciamento de eventos podem ser usados para criar formulários dinâmicos e interativos em aplicações web.
